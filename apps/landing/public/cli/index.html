TRUSIN CLI

Bring production webhooks to your local service.

The trusin CLI connects your API token, streams matching events, and forwards them to a loopback address—without exposing your machine through a public tunnel.

curl -fsSL https://download.trusin.my.id/install.sh | sh
trusin set-token ts_your_token
trusin listen --source stripe --target http://127.0.0.1:3000
Install the CLICreate an API token

What developers can do

  • Forward selected events from trusin to a local loopback service.
  • List recent events and retry failed deliveries from the terminal.
  • Open the dashboard and manage a token without storing a password.
