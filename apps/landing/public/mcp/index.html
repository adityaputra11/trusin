Give AI clients reliable webhook context.

trusin bundles an MCP server that lets supported AI clients query delivery health, metrics, events, attempts, and safely request sends or retries through your API token.

{
  "command": "trusin",
  "args": ["mcp"],
  "env": { "TERUSIN_TOKEN": "ts_your_token" }
}
Configure MCPInstall the CLI

Tools built for operations

  • get_health and get_metrics for an operational overview.
  • list_events, get_event, and get_delivery_attempts for investigation.
  • send_webhook and retry_event for controlled action.
Credentials stay with your client.

The MCP server runs over local stdio and reads TERUSIN_TOKEN. It does not use browser credentials or a default local password.

Related resources

Install the trusin CLI first, then create a token in Settings. Learn how event delivery works in Webhook delivery.
