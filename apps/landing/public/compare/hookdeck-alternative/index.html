Looking for a self-hosted alternative to Hookdeck?

Hookdeck provides a managed event gateway for receiving, routing, transforming, and recovering webhook traffic. trusin is an open-source option for teams that want to operate their webhook relay and delivery history inside their own environment.

Managed gateway or self-hosted relay?

QuestionHookdecktrusin
Primary modelManaged event gateway for inbound and outbound events.Open-source webhook relay and delivery operations stack.

When trusin is a good fit

trusin is a practical choice when you want webhook events stored alongside systems you operate, need a focused delivery timeline for engineering teams, or want a CLI and MCP server for local and AI-assisted workflows.

Evaluate the current product surface directly.

Managed gateway capabilities change over time. Confirm current pricing, regional availability, compliance needs, and feature requirements with each vendor before deciding.
